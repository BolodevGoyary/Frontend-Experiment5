# Frontend-Experiment5: Form Validation (JavaScript Form Handling)

## Objective
To design an HTML registration/contact form and implement client-side validation using JavaScript.

## Files Included
- `index.html` – Contains the form structure and input fields.
- `style.css` – Styles the form layout and appearance.
- `script.js` – Performs client-side validation for all fields.

## Features
- Input fields: Full Name, Email, Password, Confirm Password, and Message.
- Checks for:
  - Empty fields
  - Valid email format
  - Password length
  - Password–confirmation match
- Error messages shown below each field.
- Form submission prevented until all fields are valid.

## How to Run
1. Download or clone the repository:
git clone <your-repository-link>
2. Open the folder.
3. Double-click `index.html` to open it in any web browser.
4. Fill the form to test validation.

## Output
A working registration form that validates all inputs on the client-side using JavaScript.
